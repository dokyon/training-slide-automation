## 🎙️ 音声生成の使い方

### 基本的な流れ

1. **テキストを入力**
2. **ファイル名を指定**（オプション）
3. **音声を生成**
4. **ダウンロード**

### 詳細手順

#### 1. テキスト入力

```
ナレーション用テキスト欄に入力：

例：
こんにちは。これはChatGPTを使ったナレーション生成のテストです。
AIについて説明します。
```

- 文字数がリアルタイムで表示されます
- 長いテキストもOK（5000文字まで推奨）

#### 2. ファイル名指定（任意）

```
例: introduction
```

- 空欄の場合は自動で `narration_<タイムスタンプ>` になります
- 日本語も使えます

#### 3. 音声生成

**「🎙️ 音声を生成する」ボタンをクリック**

生成中：
- ローディングアニメーション表示
- ボタンが無効化

完了時に表示される情報：
- ✅ 音声生成完了！
- **ファイル名**: `introduction_1234567890.mp3`
- **文字数**: 50 文字
- **ファイルサイズ**: 125.61 KB
- **✅ ファイルの保存を確認しました**
- **💾 保存先**: フルパス表示

#### 4. ダウンロード

「⬇️ 音声をダウンロード」ボタンをクリック

## 📖 辞書管理

### 辞書とは？

専門用語や固有名詞の読み方を制御する機能

**例:**
- `ChatGPT` → `チャットジーピーティー` ✅
- `API` → `エーピーアイ` ✅
- `AI` → `エーアイ` ✅

### 登録済みワード（29語）

- ChatGPT, AI, OpenAI, LLM, GPT
- API, UI, FAQ, PREP法
- Stella, SaaS, OAuth, JWT
- Excel, Google, Microsoft, Slack
- Zapier, Make.com
- PDF, SQL, CSV
- など

### ワードを追加する

1. **「辞書管理」タブをクリック**
2. **新しいワードを追加**欄に入力

```
元のワード: Notion
読み方: ノーション
```

3. **「➕ ワードを追加」ボタンをクリック**
4. 確認メッセージが表示される

### ワードを検索する

検索ボックスにキーワードを入力すると、リアルタイムで絞り込まれます。

### ワードを削除する

1. 削除したいワードの右側の「削除」ボタンをクリック
2. 確認ダイアログで「OK」

## ⚙️ 詳細設定

### 有料プラン vs 無料プラン

#### 無料プラン
- 制限: 1日15リクエスト
- 処理時間: 約35秒/音声
- コスト: 0円

#### 有料プラン（推奨）
- 制限: なし
- 処理時間: 約3-5秒/音声
- コスト: 約2円/音声

### プランの切り替え

`.env` ファイルを編集：

```bash
# 有料プラン
GEMINI_PAID_TIER=true

# 無料プラン
GEMINI_PAID_TIER=false
```

変更後、アプリを再起動してください。

## 🐛 トラブルシューティング

### ❌ エラー: フォルダを開けませんでした

**原因:** 保存先のパスが無効

**解決方法:**
1. 保存先パスを確認
2. パスが存在するか確認
3. パスの入力ミスを確認

### ❌ エラー: 音声生成に失敗しました

**原因1:** APIキー未設定

**解決方法:**
1. `.env` ファイルを確認
2. `GEMINI_API_KEY` が設定されているか確認

**原因2:** API制限（無料プラン）

**解決方法:**
1. 翌日まで待つ
2. 有料プランに切り替える

### ❌ エラー: ファイルの保存に失敗しました

**原因:** 保存先のフォルダに書き込み権限がない

**解決方法:**
1. 保存先を別の場所に変更
2. 例: デスクトップやDocumentsフォルダ

### ⚠️ 音声の読み方が間違っている

**解決方法:**
1. 「辞書管理」タブを開く
2. 間違った読み方をされているワードを追加
3. 正しい読み方を登録
4. 再度音声生成
